🚀 Sangram Morkar — Portfolio Website
A modern and fully responsive personal portfolio website built using React, Vite, and Tailwind CSS. This website showcases my skills, projects, and contact information with a clean UI, smooth animations, and dark/light theme support.

👨‍💻 About Me
I am Sangram Morkar, a passionate Full Stack Developer who enjoys building modern web applications and solving real-world problems using technology.

🛠 Tech Stack
Frontend: React.js, HTML, CSS, Tailwind CSS, JavaScript
Routing: React Router
Build Tool: Vite
Icons: Font Awesome
Fonts: Google Fonts
📂 Features
Responsive design for all devices
Dark / Light theme toggle
Smooth animations and modern UI
Project showcase section
Skills section
Contact form
Clean and component-based architecture
